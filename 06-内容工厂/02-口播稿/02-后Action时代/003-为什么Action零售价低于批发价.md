# 第一部分
## 15秒黄金开头

你有没有发现一个很扎心的现象？

有些商品，Action的零售价，可能比传统批发渠道给你的批发价还低。

这不是简单的价格问题。

这是欧洲供应链效率正在重构。

# 第二部分
## 建立Leon权威

我在美的负责欧洲渠道开发的时候，长期和Carrefour、Auchan、Lidl这些大客户打交道。

我很清楚，大客户谈价格，从来不是只看一个单品报价。

他们看的是你的供货稳定性、交付能力、质量控制、售后风险，还有长期合作成本。

后来我在AIGOSTAR做西班牙市场，建立欧洲本地供应链，也更深刻感受到：欧洲零售的价格，背后其实是一整套系统。

# 第三部分
## 商业案例

我们来看Action为什么能便宜。

很多人以为，它只是把毛利压低。

不是这么简单。

Action在欧洲有接近3000家门店，SKU大约6000个。

这意味着什么？

它不是每个品类都铺得很散，而是把有限SKU做出更大的采购量。

采购集中，单品量大，物流标准化，门店陈列标准化，商品周转快，库存停留时间短。

这些因素叠加起来，单个商品的综合成本就降下来了。

而传统批发链条呢？

采购分散，层级较多，SKU复杂，仓储成本高，动销慢。

所以你看到的不是一个价格差，而是两个系统之间的效率差。

这也是为什么有些批发商会觉得很难受：我已经没赚多少了，为什么它还能卖得比我便宜？

答案是，它不是靠一个环节便宜，而是整个系统更高效。

# 第四部分
## 未来趋势

未来欧洲批发行业会面临一个很大的变化。

批发商如果还只是“我有货，你来拿”，会越来越难。

因为零售商要的不只是货，而是能不能卖得动。

未来批发商要提供选品逻辑、价格建议、陈列方案、促销节奏、动销反馈。

换句话说，批发商要从供货商，升级为门店运营服务商。

# 第五部分
## 价值升华

所以，不要只问Action为什么便宜。

更要问：我们的供应链哪里低效？我们的SKU哪里太散？我们的库存哪里太慢？我们的门店有没有动销能力？

后Action时代，真正的低价，不是压出来的。

是效率长出来的。

未来供应链拼的不是谁报价更狠，而是谁能帮助门店更快赚钱。
