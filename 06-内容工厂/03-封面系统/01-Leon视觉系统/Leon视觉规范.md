# Leon视觉规范

# 定位

Leon视觉系统用于统一LEROS全部短视频封面、文章封面、白皮书封面和课程封面。

核心识别：

- Leon讲欧洲零售
- Leon讲后Action时代
- Leon讲欧洲华人百货升级
- Leon讲中国企业出海欧洲

# 人物位置

Leon人物默认位于画面右侧，占画面宽度约30%-40%。

诊断类封面可放左侧，但同一系列内不频繁变化。

# 人物角度

默认使用正面半身，身体微微朝向标题区域。

门店诊断、货架分析、数据分析类内容，可使用侧身指向背景元素。

# 服装

固定使用深色西装、深色商务外套或深色商务休闲夹克。

内搭以白色、浅灰、深蓝为主。

避免过度休闲、花色、强娱乐化服装。

# 表情

表情应保持严肃、冷静、专业、有判断力。

不使用夸张大笑、惊讶、娱乐化表情。

# 占比

竖版1080x1920封面中，Leon高度建议占画面55%-70%。

人物头部不贴边，肩部完整，手势不遮挡主标题。

# 固定识别语

左上角固定栏目识别：

`LEROS | 欧洲零售研究`

具体系列可替换为：

- `LEROS | 后Action时代`
- `LEROS | 华人百货升级`
- `LEROS | Go Europe`
- `LEROS | 猎鹰计划`
