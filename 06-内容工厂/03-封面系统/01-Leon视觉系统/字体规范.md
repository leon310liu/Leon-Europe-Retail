# 字体规范

# 主标题字体

建议：

- 思源黑体 Heavy
- 阿里巴巴普惠体 Heavy
- 站酷高端黑

要求：

- 粗
- 短
- 有冲击力
- 5字以内优先

# 副标题字体

建议：

- 思源黑体 Medium
- 阿里巴巴普惠体 Medium

要求：

- 解释主标题
- 10字以内优先
- 不抢主标题视觉中心

# 警示字体

建议：

- 粗体白字红底
- 红色描边白字
- 红色警示标签

适用词：

- 危险
- 变天
- 失效
- 倒挂
- 淘汰
- 警醒

# 数据字体

建议：

- DIN Condensed
- Bebas Neue
- 阿里巴巴普惠体 Bold

适用：

- 6000 SKU
- 138亿欧元
- 2918家门店
- 未来十年
- 2万-5万SKU

# 字体层级

主标题最大。

副标题次之。

底部价值点最小。

不要让封面出现超过三种字体风格。
