# C. Leon个人IP视觉规范V1

## 人物位置规范

- Leon人物固定在画面右侧。
- 使用半身商务形象，身体可略朝左，视线保持冷静、思考感。
- 人物高度占竖版画面55%-70%。
- 人物不遮挡主标题，不压住底部价值点。

## 标题规范

- 主标题最大字号，优先使用4-6个字的战略关键词。
- 本条主标题：`后Action时代`。
- 副标题用于解释时代变化，不抢主标题。
- 本条副标题：`欧洲零售已经变天`。
- 顶部栏目固定：`LEROS｜后Action时代`。

## 色彩规范

- 主色：深蓝黑 `#0B1220` / `#111827`。
- 警示色：危机红 `#E11D48`。
- 效率强调色：效率黄 `#FACC15`。
- 正文字色：冷白 `#F8FAFC`。
- 辅助线条：钢灰 `#64748B`。

## 字体规范

- 主标题：思源黑体 Heavy / 阿里巴巴普惠体 Heavy / 站酷高端黑。
- 副标题：思源黑体 Medium / 阿里巴巴普惠体 Medium。
- 价值点：中粗黑体，手机端一眼可读。
- 英文IP标识：Montserrat / Helvetica / DIN。

## 图标规范

- 价值点统一使用`✓`。
- `✓`使用效率黄或冷白，不使用复杂图标。
- 不使用娱乐化贴纸、表情包、爆炸形状。

## LOGO规范

- 顶部栏目视为内容LOGO：`LEROS｜后Action时代`。
- 右下角固定个人IP：`Leon｜欧洲零售研究`。
- 未来可扩展英文版：`Leon · Europe Retail`。
- LOGO不宜过大，作用是统一识别，不抢标题。

## 每条视频固定元素

- 深蓝商业背景。
- Leon右侧半身商务形象。
- 顶部`LEROS｜系列名`。
