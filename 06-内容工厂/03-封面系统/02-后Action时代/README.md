# Leon个人视觉系统V1

本目录用于管理《后Action时代》系列封面系统。

目标不是做单条视频封面，而是建立Leon在欧洲零售领域的长期个人IP视觉识别。

观众未来看到以下组合，应立即识别为Leon的欧洲零售内容：

`深蓝商业背景 + Leon本人半身形象 + 黄红警示色 + 欧洲零售纪实风`

# 01 核心定位

Leon的第一视觉身份不是情绪博主，也不是单纯评论者，而是：

- 欧洲零售研究者
- 20年欧洲零售从业者
- 行业变化观察者
- 中国企业出海欧洲的商业顾问
- 欧洲华人百货升级研究者

封面气质应接近：

- Netflix商业纪录片
- 行业白皮书
- 商业咨询顾问
- 战略研究报告

关键词：冷静、克制、专业、有思考感、有历史纵深。

# 02 固定封面结构

```text
顶部：LEROS｜系列名
中部：战略主标题
标题上方：红色预警标签
标题下方：解释型副标题
左下：三条价值点
右侧：Leon本人半身商务形象
右下：Leon个人IP标识
```

# 04 人物位置规范

- Leon固定在画面右侧。
- 使用本人半身商务形象。
- 表情冷静、克制、专业、有判断力。
- 人物占画面高度55%-70%。
- 不遮挡主标题和价值点。
- 未来100条视频尽量保持同一视觉位置，形成记忆锚点。

# 05 标题规范

- 主标题必须是第一视觉层级。
- 主标题使用最大字号，优先4-6个字。
- 副标题用于解释主标题，不与主标题抢视觉。
- 顶部栏目固定小字号，不喧宾夺主。
- 本系列不使用夸张标题党语气。

# 06 色彩规范

- 主色：深蓝黑 `#0B1220` / `#111827`
- 警示色：危机红 `#E11D48`
- 效率色：效率黄 `#FACC15`
- 正文字：冷白 `#F8FAFC`
- 辅助线：钢灰 `#64748B`

色彩原则：深蓝建立专业感，红色建立行业预警，黄色强调效率和机会。

# 07 字体规范

- 主标题：思源黑体 Heavy / 阿里巴巴普惠体 Heavy / 站酷高端黑
- 副标题：思源黑体 Medium / 阿里巴巴普惠体 Medium
- 价值点：中粗黑体，保证手机端可读
- 英文标识：Montserrat / Helvetica / DIN

不超过三种字体风格。

# 08 图标规范

- 价值点统一使用`✓`。
- `✓`可使用效率黄或冷白。
- 不使用娱乐化贴纸、夸张箭头、爆炸图形、表情包。
- 图标服务于阅读，不制造噪音。

# 09 LOGO规范

- 顶部栏目是内容LOGO：`LEROS｜后Action时代`。
- 右下角是个人IP标识：`Leon｜欧洲零售研究`。
- 个人IP标识未来100条视频保持统一位置。
- LOGO不宜过大，主要作用是建立识别，不抢主标题。

# 10 禁止事项

禁止：

- 娱乐化
- 自媒体鸡汤风
- 廉价营销风
- 大面积文字堆积
- 夸张表情
- 过度促销感
- 让观众第一眼只看到非核心词

# 11 未来扩展原则

未来100条视频封面统一遵循：

`系列栏目 + 战略主标题 + Leon人物 + 三条价值点 + 个人IP标识`

所有封面先建立专业权威，再考虑传播效率。
