# 封面风格

商业诊断风 + 库存危机风。重点表现“货多变成风险”。

# 背景图

密集货架、仓库库存、堆满商品的百货通道，叠加SKU数字和库存警示图标。

# Leon位置

左侧半身，双手交叉或一手指向库存，表现判断与诊断。

# 主标题

货多危险

# 副标题

SKU越多越要小心

# 色彩方案

暗灰 + 红色风险框 + 白色大字 + 橙黄色库存数字。避免花哨，保持压迫感。

# 情绪

扎心、焦虑、危机、清醒。
