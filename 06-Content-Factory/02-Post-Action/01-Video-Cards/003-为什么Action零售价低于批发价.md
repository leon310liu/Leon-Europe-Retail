# 核心观点

Action的低价不是简单压供应商，而是规模采购、精选SKU、高周转和标准化运营共同形成的系统结果。

# 认知冲击点

当批发商发现自己的批发价比Action零售价还高时，问题不只是采购价，而是整个供应链效率和渠道结构正在被重构。

# 商业逻辑

传统批发链条层级多、采购分散、SKU复杂、库存周转慢。Action依靠大规模门店、集中采购、统一物流和高周转，把单品成本、库存成本和运营成本压低。它不是单品价格优势，而是系统效率优势。

# Leon真实经历

Leon在美的时期开发欧洲主流渠道，深知Carrefour、Auchan、Lidl等大客户看重的不只是单次报价，而是稳定供货、质量控制、交付能力、风险管理和长期合作成本。AIGOSTAR时期建立欧洲供应链体系，也理解本地仓储、渠道和价格体系的复杂性。

# 欧洲案例

Action依靠欧洲多国门店规模和约6000个高效SKU形成采购集中度，同时保持高频上新和快速淘汰。相比传统批发渠道，它能用更少SKU形成更强采购量，用更快周转摊薄运营成本。
