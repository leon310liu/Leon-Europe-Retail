# 第一部分
## 15秒黄金开头

看到Action低价之后，很多老板第一反应是：那我也降价。

但我想提醒你一句：

低价不是战略。

没有系统效率支撑的低价，可能是利润自杀。

# 第二部分
## 建立Leon权威

我在美的做欧洲渠道开发时，接触过很多欧洲大型零售客户。

这些客户当然关心价格，但他们绝对不只看价格。

他们还看质量稳定不稳定，交付准不准，认证齐不齐，售后风险高不高，供应商能不能长期合作。

后来我在AIGOSTAR做西班牙市场，也更清楚地看到：价格背后其实是运营能力。

# 第三部分
## 商业案例

Action便宜，很多人只看到结果。

但结果背后是什么？

是集中采购，是有限SKU，是高周转，是标准化门店，是集中物流，是快速淘汰。

这些东西一起作用，才支撑了它的低价。

如果一个普通门店只是看到它便宜，然后跟着降价，会发生什么？

毛利先下降。

库存周转没有变快。

采购成本没有下降。

陈列效率没有提高。

顾客体验也没有明显改善。

最后就变成：价格低了，利润薄了，老板更累了，但竞争力没有真正提高。

这就是很多门店打价格战最危险的地方。

你学到的是Action的表面，没有学到Action的系统。

# 第四部分
## 未来趋势

后Action时代，低价会继续存在，但低价的门槛会越来越高。

未来真正能做低价的企业，一定要有强供应链、高周转、低库存、标准化运营和稳定现金流。

对华人百货来说，不是不能做价格竞争，而是要先算清楚：

哪些商品可以做引流？

哪些商品必须保利润？

哪些商品是形象品？

哪些商品根本不该再进？

# 第五部分
## 价值升华

所以我的建议很简单：

不要学Action的低价，要学Action的效率。

低价只是结果，效率才是原因。

如果你的门店没有提高周转，没有优化SKU，没有降低库存，没有改进陈列，只是降价，那不是战略，那是把自己的利润送出去。

未来零售拼的不是谁更敢便宜。

而是谁便宜之后，还能健康赚钱。
