# 第一部分
## 15秒黄金开头

欧洲消费者已经被Action重新教育过了。

很多老板还以为，顾客只是变穷了、只想买便宜货。

不完全是。

真正的变化是：顾客变得更会比较，也更懂得选择。

# 第二部分
## 建立Leon权威

我在西班牙和欧洲零售一线观察了很多年。

在AIGOSTAR时期，我不仅做渠道，也做品牌、陈列和营销宣传。

我很清楚，一个商品能不能卖，不只是价格问题。

顾客能不能看懂，能不能感受到价值，能不能快速决定购买，这些都会影响销售。

# 第三部分
## 商业案例

Action做了一件很重要的事：

它让欧洲消费者相信，低价也可以有品质，便宜也可以有惊喜。

你走进Action，商品不一定贵，但它经常有新东西。

有些商品价格很低，但包装、陈列、场景感都让人觉得“可以买一个试试”。

这就是它厉害的地方。

它不是单纯卖便宜货，而是不断制造购物惊喜。

再看很多传统百货，货也不少，价格也未必贵。

但问题是，商品放在那里，顾客看不懂；品类堆在一起，顾客找不到；没有内容，没有场景，也没有上新的节奏。

顾客不是不买。

顾客是不知道为什么要买。

# 第四部分
## 未来趋势

欧洲经历通胀和生活成本压力之后，消费者会越来越理性。

但理性不等于不消费。

他们会更看重性价比，更看重使用场景，更看重购买效率。

未来门店不能只问：我的东西便不便宜。

还要问：顾客看得懂吗？觉得值吗？愿意拍给朋友看吗？愿意下次再来吗？

这就是后Action时代的消费者变化。

# 第五部分
## 价值升华

所以，华人百货不是没有机会。

我们离社区近，有很多实用商品，也有灵活采购能力。

但必须学会重新表达商品。

用更清楚的陈列，用更强的场景，用更快的上新，用短视频和社群告诉顾客：这个东西为什么值得买。

顾客不是没钱。

顾客只是比以前更聪明了。

谁能先理解顾客，谁就还有机会。
