# 封面风格

商业咨询风 + 新闻预警风。画面要像一张欧洲零售行业研判封面，不要做成娱乐化短视频海报。

# 背景图

欧洲Action门店外景或欧洲折扣零售门店入口，叠加轻微数据网格和红色趋势箭头。

# Leon位置

右侧半身，正面偏左看镜头，穿深色西装或商务休闲外套，表情严肃、判断感强。

# 主标题

变天了

# 副标题

后Action时代

# 色彩方案

深蓝黑背景 + Action红色警示线 + 白色主标题 + 少量黄色强调。整体要高级、冷静、有商业危机感。

# 情绪

震撼、警醒、行业预判、权威感。
