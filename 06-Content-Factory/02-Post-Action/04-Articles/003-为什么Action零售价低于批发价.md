# 开头观点

为什么Action的零售价有时会低于传统批发渠道的批发价？这个现象让很多欧洲华人批发商和零售商感到困惑，甚至感到无力。表面看，这是价格竞争；深层看，这是供应链效率、渠道结构和商品组织方式之间的差距。

价格倒挂不是偶然现象，也不是简单靠压低毛利就能解释。它背后反映的是：传统批发链条的成本结构和周转方式，正在被Action这类高效率零售系统挑战。

# 行业现状

欧洲非食品折扣零售进入规模化和系统化阶段。Action、TEDi、Lidl、Aldi等渠道用不同方式提高供应链效率。消费者看到的是低价，行业内部看到的应该是采购集中、SKU精简、物流标准化、库存周转和现金流能力。

传统批发体系的问题在于链条较长、采购分散、SKU复杂、仓储成本高、终端动销慢。一件商品从工厂到进口商、批发商、零售商，再到消费者，中间每个环节都要承担库存、资金和运营成本。如果终端动销慢，成本就会继续被放大。

# Leon真实经历

Leon在美的时期负责欧洲渠道开发，长期与Carrefour、Auchan、Lidl等欧洲主流零售客户打交道。他深知大型客户谈价格，不只是看一个单品报价，而是看供货稳定性、质量控制、交付能力、售后风险和长期合作成本。

AIGOSTAR时期，Leon参与欧洲本地供应链建设，也管理过8000多个SKU。这个经历让他理解：价格不是孤立数字，而是一套供应链系统的结果。仓储、补货、周转、陈列、渠道层级，每一项都会影响最终零售价。

# Action案例

Action的低价来自系统效率。它以有限SKU形成更大的单品采购量，通过门店规模强化采购能力，通过标准化物流降低运营复杂度，通过快速周转降低库存停留成本。它不是每个商品都铺得很散，而是集中资源做更有效率的商品池。

公开报道显示，Action 2024年销售额约138亿欧元，2025年继续增长至约160亿欧元。这样的规模意味着它可以在部分品类上获得传统分散批发难以匹配的采购条件。更关键的是，它不是单次采购便宜，而是整个系统围绕高周转运转。

# 华人百货问题

华人批发行业过去的优势，是商品丰富、反应灵活、服务华人零售网络能力强。但今天的问题也很明显：部分批发商仍停留在“我有货，你来拿”的供货模式，没有为门店提供足够的动销支持。

当零售商发现Action零售价比批发价还低时，批发商不能只怪对方规模大。更应该反思：自己的SKU是否过散？是否有动销数据？是否帮助门店做陈列和促销？是否建立爆品测试机制？是否通过集中采购提升议价能力？

# 未来趋势

未来欧洲批发商会分化。一类批发商继续做库存搬运，利润会越来越薄；另一类批发商会升级为门店运营服务商，提供选品、陈列、价格策略、促销节奏和动销反馈。

零售商也会改变采购标准。过去门店问批发商：“这个货多少钱？”未来会问：“这个货能不能卖动？怎么陈列？毛利是多少？多久周转？是否能做内容传播？”这会倒逼批发商从供货思维转向运营思维。

# 升级建议

第一，批发商要建立SKU分层。不是所有商品都值得长期备货，必须区分爆品、利润品、季节品和淘汰品。

第二，建立门店反馈机制。批发商不能只看出货量，还要看终端动销。

第三，推动爆品共创。通过小批量测试、门店反馈、快速复购来验证商品，而不是一次性压货。

第四，提供陈列和促销建议。批发商未来的价值，不只是仓库里有什么，而是能不能帮助门店赚钱。

# 总结

Action零售价低于批发价，本质是系统效率超过传统链条效率。对华人批发商来说，这不是简单价格战，而是商业模式升级信号。未来能活得好的批发商，不是货最多的批发商，而是最懂选品、最懂动销、最能帮助门店提升经营效率的批发商。
