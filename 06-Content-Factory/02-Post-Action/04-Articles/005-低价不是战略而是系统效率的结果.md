# 开头观点

低价不是战略，而是系统效率的结果。很多传统门店看到Action低价，第一反应是跟着降价。但如果没有采购效率、库存周转、陈列标准、门店运营和现金流能力支撑，降价不是竞争策略，而是利润自杀。

后Action时代，低价会继续存在，但低价的门槛会越来越高。真正有杀伤力的低价，背后一定是系统能力；没有系统的低价，只会让门店更累、利润更薄、库存更难处理。

# 行业现状

欧洲消费者更理性，价格敏感度确实提高。但这不代表消费者只买最便宜的商品。消费者要的是综合价值：价格合理、品质稳定、购买方便、使用场景清楚、购物体验高效。

折扣零售的崛起让很多门店误以为，未来竞争就是价格战。实际上，Lidl、Aldi、Action这类企业的低价能力，来自规模采购、供应链控制、SKU精简、门店标准化和高周转。它们不是先低价再找效率，而是先有系统效率，低价才成立。

# Leon真实经历

Leon在美的时期做欧洲渠道开发，长期面对欧洲主流零售客户。他看到的真实情况是，大客户当然关心价格，但价格只是入场条件之一。质量稳定、认证合规、交付准时、售后风险、长期供货能力，都会影响客户最终判断。

AIGOSTAR时期，Leon在西班牙市场做本地运营，也更清楚地看到：一个商品最终能卖多少钱，不只由采购价决定，还由库存成本、仓储效率、门店陈列、促销节奏和渠道利润结构决定。

# Action案例

Action的低价不是孤立动作。它通过约6000个SKU控制复杂度，通过规模采购提升议价能力，通过标准化门店降低运营成本，通过高周转提升现金流效率。低价只是顾客看到的表面，背后是完整系统。

2025年Action销售继续增长至约160亿欧元，但2026年也出现同店销售放缓和利润率压力。这说明即使是高效率企业，也要不断平衡低价、增长和利润。传统门店如果只模仿低价，而没有系统能力，风险会更大。

# 华人百货问题

很多华人百货老板看到竞争对手低价，就用降价回应。但问题是，采购成本没有下降，库存周转没有变快，门店陈列没有改善，顾客体验没有提升。结果是价格低了，利润薄了，老板更累了，竞争力却没有真正提高。

低价最危险的地方，是它会掩盖经营问题。门店卖不动，可能不是因为价格贵，而是商品结构不对、陈列不清楚、顾客不知道为什么买、低效库存太多。如果没有先诊断这些问题，直接降价只是把利润送出去。

# 未来趋势

未来低价仍然重要，但低价会和效率绑定。能长期低价的企业，一定有强供应链、高周转、低库存、标准化运营和稳定现金流。不能长期低价的企业，如果硬打价格战，会被拖入低毛利陷阱。

华人百货未来要建立价格分层：引流品可以低价，利润品必须保毛利，形象品要提升门店认知，季节品要控制节奏，淘汰品要快速清理。不是所有商品都适合降价，也不是所有价格都应该跟随Action。

# 升级建议

第一，降价前先算周转。如果降价不能带来更快周转，就是牺牲利润。

第二，建立商品角色。哪些商品引流，哪些商品赚钱，哪些商品提升形象，必须清楚。

第三，先优化SKU再谈价格。低效SKU太多时，降价解决不了结构问题。

第四，提升陈列和内容表达。很多商品不是贵，而是没有被顾客理解。

第五，批发商要提供价格和动销建议，不能只给进货价。

# 总结

低价不是战略，效率才是战略。Action真正值得学习的不是便宜，而是它如何用系统支撑便宜。华人百货不应该盲目跟随价格战，而应该从SKU、库存、陈列、供应链和现金流出发，建立自己的健康低价能力。能低价又能赚钱，才是真正的竞争力。
