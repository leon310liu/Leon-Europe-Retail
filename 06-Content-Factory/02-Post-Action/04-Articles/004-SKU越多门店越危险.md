# 开头观点

SKU越多，门店越危险。这句话对传统华人百货老板来说可能很刺耳，因为过去很多门店的核心优势就是“货全”。顾客需要什么，店里最好都有；商品越多，看起来越有规模，越有安全感。

但后Action时代，SKU数量本身已经不再是优势。真正有价值的是有效SKU。低效SKU越多，门店现金流越危险，货架效率越低，管理复杂度越高。

# 行业现状

欧洲零售正在从“货多竞争”转向“效率竞争”。消费者不再愿意在混乱货架中慢慢寻找商品，而是希望快速找到、快速理解、快速判断价格是否值得。Action这样的折扣零售并不是靠无限SKU赢，而是靠精选SKU、高频上新、快速淘汰和集中采购赢。

传统华人百货常见SKU可能达到两万到五万个。数量看起来庞大，但如果没有动销数据、库存结构和淘汰机制，SKU越多，越容易拖累经营。每个SKU都占用采购资金、仓储空间、货架位置、员工时间和清仓成本。

# Leon真实经历

Leon在AIGOSTAR西班牙市场管理过8000多个SKU。这个经历让他非常清楚，SKU不是一个简单数字，而是一套复杂管理系统。一个SKU进入企业之后，后面跟着采购、订货、仓储、条码、价格、陈列、促销、售后和淘汰。

当SKU数量上升，如果没有系统化管理，问题会指数级增加。很多老板以为自己是在扩大商品丰富度，实际上可能是在扩大库存风险和管理成本。

# Action案例

Action约6000个SKU，但它通过动态筛选形成高效商品池。它不是商品少，而是把商品组织得更有效率。低效商品会被淘汰，新品不断测试，爆品快速放大，季节品有明确节奏。

这对华人百货最大的启发是：零售不是把尽可能多的商品摆进门店，而是让每一组商品承担清晰任务。一个SKU要么带来流量，要么带来利润，要么提升形象，要么满足季节需求。如果什么作用都不明显，就应该被重新评估。

# 华人百货问题

很多华人百货的问题不是缺货，而是货太多、太杂、太慢。库存里长期不动的商品，占用了现金流；货架上重复度高的商品，降低了顾客选择效率；缺少淘汰机制，让门店越来越像仓库。

更严重的是，低效SKU会掩盖真正好卖的商品。好商品被淹没在大量杂乱商品中，顾客看不到重点，员工也很难维护陈列。门店看起来很满，但销售效率并不高。

# 未来趋势

未来欧洲华人百货一定会进入SKU治理阶段。不是所有门店都要把SKU砍到Action一样少，但所有门店都必须知道：哪些商品真正贡献销售，哪些商品贡献利润，哪些商品只是长期占用资金。

门店管理会从经验采购转向数据采购，从一次性进货转向持续验证，从“老板觉得好卖”转向“动销结果证明好卖”。SKU治理会成为门店升级的第一道门槛。

# 升级建议

第一，建立SKU分层。至少分为爆品、利润品、形象品、季节品和淘汰品。

第二，建立动销周期。超过一定时间不动的商品，必须进入降价、组合销售或淘汰流程。

第三，控制新品测试规模。新品不要一次压太多，而是小批量测试、快速反馈、快速复购。

第四，优化货架资源。货架不是仓库，黄金位置应该给高效率商品。

第五，批发商也要参与SKU治理。批发端如果不优化，零售端会继续被低效库存拖累。

# 总结

SKU越多，不一定越安全。真正安全的是现金流健康、商品结构清晰、补货节奏稳定、顾客购买效率高。后Action时代，华人百货升级的第一刀往往不是装修，而是SKU治理。门店要从“货很多”转向“货有效”，这才是未来竞争力。
