# 10-素材库

## 定位

本目录是 Leon 欧洲零售研究的长期媒体资产库。

所有图片、视频、logo、门店素材、B-roll、品牌资料、行业背景素材，统一进入本目录管理。

## 一级结构

```text
10-素材库
├── 00-Leon原创素材库
├── 01-公共素材库
├── 02-品牌素材库
├── 03-行业素材库
├── 04-素材索引与下载清单
└── 05-版权与授权记录
```

## 00-Leon原创素材库

用于保存 Leon 本人拍摄或持有版权的素材。

包括：Leon人物口播、门店现场、货架陈列、价格标签、收银台、排队、补货、仓库物流、猎鹰行动、门店改造、老板访谈。

以后 Leon 主动发送素材时，Codex 按描述自动归类。

## 01-公共素材库

用于保存可公开视频使用的公共 B-roll，包括 CBD高楼大厦、航运港口集装箱、航空机场飞机、车水马龙交通、人来人往商业街、疯抢商品排队、仓储物流卡车、数据动画财经新闻、会议办公商务、欧洲街景商圈。

## 02-品牌素材库

用于保存大型连锁零售品牌素材。

重点品牌：Action、MRDIY、Pepco、Normal、Mercadona、TEDI、Primark、Lidl、Aldi、Costco、IKEA。

每个重点品牌统一保留：logo、门头、陈列货架、促销爆品、收银台排队、财报新闻。

## 03-行业素材库

用于保存行业解释素材，包括百货、折扣店、超市、供应链、物流仓储、跨境出海、欧洲经济、消费降级、消费升级。

## 04-素材索引与下载清单

用于记录已经检索到的素材来源。

## 05-版权与授权记录

用于记录素材来源、授权状态、可商用状态和使用风险。

公开视频优先使用：Leon原创素材、官方素材、可商用公共素材、可追溯授权素材。

新闻截图、社媒截图、Google图片搜索结果只作为研究参考，不默认用于公开视频。
