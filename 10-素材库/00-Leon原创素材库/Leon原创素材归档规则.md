# Leon原创素材归档规则

## 01 自动分类原则

Leon 主动发送的图片或视频，优先判断其内容场景，再归入对应目录。如果 Leon 已经说明素材类型，以 Leon 的描述为准。

示例：

- “这是Action门店素材” -> 02-门店现场/Action门店
- “这是猎鹰行动素材” -> 06-项目素材/猎鹰行动
- “这是改造前后素材” -> 06-项目素材/门店改造
- “这是人物口播” -> 01-Leon人物/口播
- “这是补货素材” -> 03-零售运营/补货

## 02 固定分类

01-Leon人物：口播、采访、生活。

02-门店现场：Action门店、MRDIY门店、Pepco门店、Normal门店、Mercadona门店、华人百货、葡萄牙百货。

03-零售运营：货架、堆头、促销牌、价格标签、爆品、收银台、排队、补货、顾客购物。

04-供应链现场：仓库、物流中心、集装箱、停车场。

05-城市商业：商业街、商场、购物中心、欧洲城市、欧洲农村、高速公路。

06-项目素材：猎鹰行动、门店改造、老板访谈、案例复盘。
