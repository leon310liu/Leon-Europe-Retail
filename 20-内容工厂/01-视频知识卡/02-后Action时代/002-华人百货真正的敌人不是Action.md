# 核心观点

华人百货真正的敌人不是Action，而是低周转、高库存、低效率的旧经营模式。

# 认知冲击点

Action开在旁边只是表面压力，真正危险的是门店内部的商品结构、库存结构和运营方式已经不适合后Action时代。

# 商业逻辑

传统华人百货的优势是商品丰富、社区覆盖强、老板决策快；但如果没有数据、陈列、SKU淘汰和补货机制，这些优势会变成库存压力。Action用高效率系统放大低价优势，而低效率门店即使降价，也很难真正获得竞争力。

# Leon真实经历

Leon长期深入欧洲华人百货和西班牙百元店体系，接触大量零售商、批发商和品牌商。AIGOSTAR时期既做渠道开发，也做陈列升级和营销宣传，清楚许多门店不是没有货，而是货太多、货太慢、货架没有销售逻辑。

# 欧洲案例

欧洲折扣零售和专业零售持续扩张，消费者已经习惯低价、清晰分类、快速购物和高频上新。传统杂货式门店如果仍依靠“货多、品杂、老板经验”，会被更高效的连锁系统持续挤压。
