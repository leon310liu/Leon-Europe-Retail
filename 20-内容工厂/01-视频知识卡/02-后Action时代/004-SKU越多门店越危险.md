# 核心观点

后Action时代，SKU数量不是优势，低效SKU越多，门店现金流越危险。

# 认知冲击点

很多老板以为货越全越有竞争力，但在高库存、低周转环境下，货越多可能意味着资金越沉、管理越乱、陈列越弱。

# 商业逻辑

SKU不是越多越好，而是要形成销售效率。每一个SKU都占用采购资金、仓储空间、货架位置、补货精力和清仓成本。如果没有分层管理和淘汰机制，SKU会从“商品资产”变成“现金流黑洞”。

# Leon真实经历

Leon在AIGOSTAR时期管理8000+SKU，深知SKU越多，对采购计划、库存管理、仓储协同、陈列标准、价格体系和团队执行要求越高。没有系统的SKU管理，规模越大，复杂度越高。

# 欧洲案例

Action约6000个SKU形成高效率商品池，而传统华人百货常见SKU可能达到20000到50000个。两者差异不是商品数量，而是商品效率。Action的逻辑是持续筛选、快速淘汰、集中采购和高频上新。

# 视频结构

- 15秒黄金开头：SKU越多，门店越危险。
