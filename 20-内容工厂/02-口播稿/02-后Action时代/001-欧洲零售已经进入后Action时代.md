# 第一部分
## 15秒黄金开头

欧洲零售已经变天了。

很多华人百货老板现在还以为，Action只是一个卖便宜货的店。

但我告诉你，真正可怕的不是它便宜，而是它把欧洲零售的竞争标准，整体提高了一遍。

这就是我说的：欧洲零售已经进入后Action时代。

# 第二部分
## 建立Leon权威

我为什么敢这么讲？

因为我在欧洲零售行业做了接近20年。

我在美的负责过欧洲渠道开发，接触过Carrefour、Auchan、Leroy Merlin、MediaMarkt、Lidl这些欧洲主流零售客户。

后来在AIGOSTAR，我做了约8年西班牙国家总经理和股东，管理过8000多个SKU，做过渠道开发、陈列升级、供应链建设，也长期接触欧洲华人百货和批发体系。

所以我不是从新闻里看Action，我是从渠道、门店、供应链和顾客变化里，看到了这个时代的变化。

# 第三部分
## 商业案例

我们看Action。

2024年，Action销售额大约138亿欧元，门店接近2918家，一年新增约352家门店。

这不是一家普通便宜店的增长速度。

它真正厉害的地方，是用大约6000个SKU，做到高周转、集中采购、标准化陈列、高频上新和内容传播。

再看很多传统华人百货，可能有两万、三万，甚至五万个SKU。

表面看，货更多，选择更多。

但问题是：多少SKU真的在赚钱？多少SKU压在仓库里？多少SKU占着货架，却几个月不动？

Action的逻辑不是“我什么都有”，而是“我留下最有效率的商品”。

这就是差距。

# 第四部分
## 未来趋势

后Action时代，欧洲零售会出现几个变化。

第一，消费者越来越理性，更看价格、品质、体验和性价比。

第二，渠道会越来越集中，高效率连锁会继续挤压低效率门店。

第三，SKU会越来越精简，低周转商品会被淘汰。

第四，门店不再只是卖货，而是要靠陈列、内容、动线和运营效率赚钱。

所以未来不是谁货多谁赢，而是谁周转快、供应链强、表达清楚，谁赢。

# 第五部分
## 价值升华

后Action时代不是坏消息。

它只是告诉我们：过去靠经验、靠货多、靠位置的时代，正在慢慢过去。

华人百货不是没有机会。

真正的问题是，我们要不要把自己的经营方式，从经验型门店，升级成系统型门店。

看懂这件事，危机就是机会。

看不懂，Action只是第一个压力，后面还会有更多压力。
