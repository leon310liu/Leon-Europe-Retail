# 06-视频项目库

用于管理每一期视频的最终成片素材包。

## 示例项目

- 001-后Action时代
- 002-MRDIY
- 003-葡萄牙90后老板
- 004-葡萄牙老板硬刚Action

## 每个视频项目必须包含

- 01-口播稿
- 02-一句一行提词稿
- 03-SRT字幕
- 04-封面图
- 05-数据卡
- 06-视频素材
- 07-Leon原创素材
- 08-BROLL素材
- 09-发布时间
- 10-评论区互动

## 素材引用机制

每一期视频真正使用过的素材，都必须复制进入该视频项目目录，形成最终成片素材包。

## 时间轴要求

每期视频必须输出素材时间轴，例如：

00:00-00:02 统一片头

00:02-00:10 Leon口播

00:10-00:20 Action门店

00:20-00:30 MRDIY门店

并明确素材来源、原始文件名和素材库路径。
