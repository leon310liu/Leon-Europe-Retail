# 封面风格

纪录片风 + 消费洞察风。画面要表现消费者变化，而不是单纯门店促销。

# 背景图

欧洲消费者在折扣零售门店购物、拿商品比价、看价格标签，背景可有购物车和新品货架。

# Leon位置

左侧半身，侧身看向顾客和货架，像在观察市场变化。

# 主标题

顾客变了

# 副标题

不是穷，是会选

# 色彩方案

深蓝灰 + 白色标题 + 黄色价格标签 + 少量红色强调。整体更有观察感和洞察感。

# 情绪

启发、清醒、市场洞察、轻微危机感。
