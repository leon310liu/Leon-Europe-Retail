# 色彩规范

# 主色

深蓝黑：

- `#0B1220`
- `#111827`

用途：

- 商业咨询背景
- 深度研究封面
- 方法论封面
- 战略判断类内容

# 警示色

危机红：

- `#E11D48`
- `#D71920`

用途：

- 预警
- 危机
- 价格倒挂
- 低效库存
- 经验失效
- 行业淘汰

# 效率色

效率黄：

- `#FACC15`
- `#F59E0B`

用途：

- 关键词强调
- 数据标签
- 价格标签
- 机会点
- 生存公式

# 辅助色

冷白：

- `#F8FAFC`

钢灰：

- `#64748B`

增长绿：

- `#22C55E`

用途：

- 冷白用于主标题
- 钢灰用于背景线和辅助说明
- 增长绿仅用于机会、增长、升级路径

# 禁止倾向

避免纯娱乐化高饱和配色。

避免粉色、糖果色、过度渐变、过度网红风。

整体应保持欧洲商业研究、零售咨询、行业判断的专业质感。
