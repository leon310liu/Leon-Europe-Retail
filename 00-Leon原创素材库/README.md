## 归档规则

只要Leon发送任何图片或视频，必须先判断素材类型，然后归档到对应分类目录。

每个素材目录必须建立或更新 `素材说明.md`。

## 命名规则

建议格式：

`YYYYMMDD_分类_序号.扩展名`

例如：

`20260702_Action停车场_001.mp4`
